# API cadastro de veículos.

## Descrição

API REST para cadastro de veículos e proprietários.

## Tecnologias utilizadas
- Spring Boot
- Spring Data JPA
- Flyway
- Hibernate Validator
- PostgreSQL
- Lombok
- Mockito
- JUnit
- Maven
- Java 17
- Banco de Dados: PostgreSQL
- Ferramenta de documentação: Swagger
